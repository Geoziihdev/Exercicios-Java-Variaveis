# Projetos de Lógica em Java

Este repositório contém uma coleção de pequenos projetos em Java que abordam conceitos básicos de programação e lógica. Cada projeto é uma implementação prática de um conceito específico.

## Projetos

1. **Soma de Dois Números**: `SomaDeDoisNumeros`
   - Um programa simples que soma dois números fornecidos pelo usuário.

2. **Cálculo da Média**: `CalculoDaMedia`
   - Calcula a média de uma série de números.

3. **Troca de Valores**: `TrocaDeValores`
   - Demonstra como trocar os valores de duas variáveis.

4. **Conversor de Temperatura**: `ConversorDeTemperatura`
   - Converte temperaturas entre Celsius, Fahrenheit e Kelvin.

5. **Cálculo da Área do Círculo**: `AreaDoCirculo`
   - Calcula a área de um círculo a partir do raio informado.

6. **Exibição de Dados Pessoais**: `DadosPessoais`
   - Um programa que exibe informações pessoais formatadas.

7. **Perímetro de um Retângulo**: `PerimetroRetangulo`
   - Calcula o perímetro de um retângulo com base na largura e altura fornecidas.

8. **Conversão de Unidades**: `ConversaoUnidades`
   - Converte entre diferentes unidades de medida.

9. **Contador de Dígitos**: `ContadorDigitos`
   - Conta quantos dígitos estão presentes em um número.

10. **Cálculo do Salário**: `CalculoSalario`
    - Calcula o salário com base em horas trabalhadas e taxa horária.
